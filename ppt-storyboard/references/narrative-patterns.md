# 演示叙事模式参考

本文档提供经过验证的叙事模式，用于指导分镜弧线设计。

## 模式一：经典三幕（通用性最强）

适用：几乎所有场景。特别适合管理层和客户。

```
第一幕 (25%) — 建立共鸣与紧迫感
  封面 → 现状共鸣 → 痛点数据 → 转折反问

第二幕 (50%) — 展示解法与论证
  核心洞察 → 方案展开(2-4页) → "啊哈时刻"数据 → 案例/证据

第三幕 (25%) — 愿景与行动
  愿景画面 → 行动路径 → 结尾金句
```

节奏曲线：
```
情感强度
  高 │          ╱ "啊哈时刻"
     │    ╱╲  ╱              ╲   ╱ 愿景
     │   ╱  ╲╱                ╲ ╱
     │  ╱                      ╳
  低 │╱ 封面                    ╲→ 行动
     └──────────────────────────────→ 页码
      1    3    5    7    9   11  12
```

## 模式二：问题-方案（适合技术决策汇报）

适用：向技术领导层汇报技术选型、架构升级。

```
开场 → 问题定义 → 约束条件 → 方案A vs B → 推荐方案 → 风险与缓解 → 时间线 → 决策请求
```

特点：
- 不讲故事，讲逻辑
- 每页信息密度可以稍高
- 用对比页（Before/After 或 方案A/方案B）作为核心论证工具

## 模式三：英雄之旅（适合客户案例、品牌故事）

适用：客户演示、品牌故事、成功案例分享。

```
日常世界 → 遇到挑战 → 遇见向导(我们) → 获得方案 → 采取行动 → 取得成果 → 新常态
```

特点：
- 客户是英雄，我们是向导
- 用具体细节和数字让故事可信
- 前后对比形成强烈反差

## 模式四：层层揭秘（适合投资方、愿景宣讲）

适用：向投资方展示商业愿景、向全员宣讲战略。

```
一个意外的事实 → "为什么会这样？" → 表面原因 → 更深层原因 →
真正的洞察 → "我们能做什么？" → 方案 → 规模化可能 → 行动
```

特点：
- 像剥洋葱一样逐层深入
- 每一层都让观众觉得"原来如此"
- 最后的洞察让前面所有铺垫都串联起来

## 视觉母题库

### 光与暗

- **从黑暗到光明**：迷雾→晨光→正午阳光
- **聚光灯效应**：一个焦点从暗处浮现
- 适用：突破困境、创新发现、战略清晰化

### 旅程与路径

- **从起点到终点**：蜿蜒小路→宽阔大道
- **分岔路口**：选择与决策
- 适用：战略规划、路线图、转型

### 建造与生长

- **种子到大树**：自然生长
- **蓝图到建筑**：人工建造
- 适用：产品演进、团队成长、平台建设

### 尺度变化

- **微观到宏观**：一个细节→全景视野
- **从个体到群体**：一个用户→百万用户
- 适用：规模化故事、市场机会

### 解构与重组

- **拆解旧系统**：分崩离析的齿轮
- **重构新系统**：流线型的新结构
- 适用：技术重构、组织变革、流程再造

### 连接与网络

- **孤岛到桥梁**：断裂→连接
- **节点到网络**：单点→网状
- 适用：平台战略、生态系统、协作

## 页面类型节奏建议

好的节奏是"密-疏-密-疏"交替，像呼吸一样有张有弛：

| 页面类型 | 信息密度 | 情感能量 | 典型时长(秒) | 功能 |
|---------|---------|---------|-------------|------|
| 封面 | 极低 | 中 | 10-15 | 建立第一印象 |
| 金句页 | 极低 | 高 | 5-8 | 制造冲击、转折 |
| 数据页 | 中 | 高 | 15-20 | 用事实论证 |
| 内容页 | 中-高 | 中 | 30-45 | 展开论述 |
| 过渡页 | 极低 | 低 | 3-5 | 呼吸、切换 |
| 对比页 | 高 | 高 | 20-30 | 制造反差 |
| 愿景页 | 低 | 高 | 10-15 | 激发想象 |
| 行动页 | 中 | 中-高 | 15-20 | 推动行动 |
