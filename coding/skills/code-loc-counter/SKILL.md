# Code LOC Counter

统计项目的代码行数，默认排除测试文件和注释，并输出总代码行数与按语言明细。

## 流程

### Step 1: 确定扫描目标

- 如果用户给了路径，使用该路径
- 如果用户没给路径，默认扫描当前目录
- 默认排除测试文件和注释

### Step 2: 运行统计脚本

执行本技能 `scripts/` 目录下的脚本：

```bash
python3 "<本技能目录>/scripts/count_code_lines.py" $ARGUMENTS
```

常用参数：
- 无参数：扫描当前目录
- `某个路径`：扫描指定目录或文件
- `--include-tests`：把测试文件也计入统计
- `--json`：输出 JSON 结果

### Step 3: 整理结果

默认用简洁格式反馈：
- 总代码行数
- 按语言明细
- 扫描文件数
- 跳过的测试文件数
- 跳过的 unknown / unsupported 文件数

如果用户要求更细，再补充说明哪些文件或语言被跳过。
